   Magnitudes y unidades
   
  

 

FIS v.10- Sabiendo que los vectores A y B son los dados en el ejercicio 4, calcular para cada caso el vector D que cumple:

  (I) A + D = B       (II) A + B + D = F = (10 ; 10)

Ej 4. Dados los vectores A y B indicados, hallar gráficamente su suma o resultante.
    a) A = (-3; 2)    B = (-2; 5)
 

No le tenemos miedo, y vamos a operar analíticamente. Obviamente componente a componente:

(I) A + D = B

Primera componente:

xA + xD = xB

– 3 + xD = – 2

De ahí:

xD = 1

Segunda componente:

yA + yD = yB

2 + yD = 5

De ahí:

yD = 3

    
  D = (1 ; 3) (I)
   

(II) A + B + D = F = (10 ; 10)

Primera componente:

xA + xB + xD = xF

– 3 – 2 + xD = 10

De ahí:

xD = 15

Segunda componente:

yA + yB + yD = yF

2 + 5 + yD = 10

De ahí:

yD = 3

    
  D = (15 ; 3) (II)
   
Según lo que indica el ejercicio 4, hay todavía dos casos más para A y B. Te los dejo a vos.  

 
     

Desafío: Graficar.
